Robot Driving System


The system is composed of a robot (R) and a device (TRC) that can be used to send commands to the robot.

The behavior of the system can be summarized as follows:
  1. the robot R initially waits for commands
  2. when the TRC sends a command, the robot executes the command
  3. the robot R  sends information about its state to the every DT (DT>0) seconds
  4. the TRC shows to the user the received information about the robot's state 
Thus, the robot R is a proactive / reactive entity.

Commands    

The robot commands are (intially): forward, backward, stop
